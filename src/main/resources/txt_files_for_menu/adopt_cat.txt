Здесь Вы можете узнать как взять питомца из приюта.
Выберите один из вариантов запроса:

/meeting_with_cat - знакомство с питомцем в приюте:

/documents_for_adopting_cat - список документов, для оформления кошки:

/recommendations_for_transporting_cat - рекомендации по перевозке кошки:

/improvement_home_for_kitty - список рекомендаций по обустройству дома для котенка:

/improvement_home_for_adult_cat - список рекомендаций по обустройству дома для взрослой кошки:

/improvement_home_for_disabled_cat - список рекомендаций по обустройству дома для кошки с ограниченными возможностями
(зрение, передвижение):

/reasons_for_rejection_cat - список причин, почему могут отказать и не дать разрешение на оформление питомца:

/cats_volunteer - пообщаться с волонтером если есть еще вопросы
