Для указания в профиле "усыновителя" взятого им из приюта питомца,
прошу ввести следующую команду:
питомец: уникальный_номер_усыновителя; вид_питомца; кличка_питомца
например надо указать, что "усыновитель" с уникальным_номером 11111111111 взял из приюта кошку по кличке Машка,
значит вводим:
питомец: 11111111111; кошка; Машка
