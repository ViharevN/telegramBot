<h2>ТЕЛЕГРАМ-БОТ ДЛЯ ПРИЮТА ЖИВОТНЫХ.</h2>

***Телеграм-бот, который отвечает на вопросы людей,
интересующихся о том, что необходимо сделать для
того, чтобы взять из приюта какого-либо питомца.***

<h2>Backend:</h2>
Java 17,
Maven,
Spring Boot,
Spring Web,
Spring Data,
Spring JPA,
GIT,
REST,
Swagger,
Stream API,
SQL,
PostgreSQL

<h2>Front-end:</h2>
Telegram
